# Canopy Website

A modern Astro website for Canopy — growing the future of agriculture.

## Tech Stack

- **Astro 5** — Static site generator
- **TinaCMS** — Content management for the blog
- **Tailwind CSS v4** — Styling with electric blue (#00D4FF) theme
- **React** — Interactive components
- **Vercel** — Deployment

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server with TinaCMS
npm run dev

# Build for production
npm run build
```

## TinaCMS Setup

1. Copy `.env.example` to `.env`
2. Set up Tina Cloud at [app.tina.io](https://app.tina.io)
3. Add your `TINA_CLIENT_ID` and `TINA_TOKEN` to `.env`

## Deployment

Push to the `main` branch and Vercel will automatically deploy.
